# 技能名称：日语朗读作业批改（教师/学生双模式）

## 基本信息
- 技能ID：japanese-reading-grader
- 版本：2.1.0
- 适用场景：
  1. 课文朗读作业批改
  2. 句子朗读作业批改
  3. 单词朗读作业批改
  4. 替换练习操练
  5. 教师批量批改 / 学生自主自测
  6. 学生历史成绩追踪与能力分层
  7. 学生自适应学习
- 目标用户：高校日语教师、日语学习者

## 运行说明
本技能使用 Whisper medium 模型进行本地语音转写。
固定使用 medium 模型，不切换大小，适配笔记本电脑，避免爆显存、卡顿或崩溃。
本技能支持课文朗读、句子朗读、单词朗读、替换练习四种发音作业，自动适配教师批改与学生自测双模式。

## 支持练习类型（自动识别）
- 单词朗读：单个或多个单词朗读
- 句子朗读：句型例句朗读
- 课文朗读：长篇课文完整朗读
- 替换练习：按要求替换词汇后朗读句子

## 学生水平分层（N5～N1）

### 等级定义
| 等级 | 能力描述 | 适用场景 |
|------|----------|----------|
| N5 | 入门级，能读假名和简单单词 | 初级教材第1-10课 |
| N4 | 基础级，能读短句和简单课文 | 初级教材第11-20课 |
| N3 | 进阶级，能读中长句和日常对话 | 中级教材前半 |
| N2 | 准高级，能读复杂句和短文 | 中级教材后半 |
| N1 | 高级，能读长文和自然语速 | 高级教材/新闻 |

### 各层级评分差异
| 等级 | 发音容忍度 | 语调要求 | 流畅度要求 | 纠错粒度 |
|------|-----------|----------|-----------|----------|
| N5 | 宽松（清浊/长音小错不扣分） | 基本升降调即可 | 允许较多卡顿 | 只纠致命错误 |
| N4 | 中等（明显错误扣分） | 句尾调基本正确 | 允许少量停顿 | 纠主要错误 |
| N3 | 较严（促音/长音严格） | 句中升降调关注 | 较流畅 | 纠大部分错误 |
| N2 | 严格（接近母语标准） | 自然语调 | 无明显停顿 | 纠所有错误 |
| N1 | 极严（细微差别也扣分） | 情感/强调语调 | 完全自然 | 精细纠错 |

### 层级扣分系数
| 错误类型 | N5 | N4 | N3 | N2 | N1 |
|----------|----|----|----|----|----|
| 促音错误 | 0.3 | 0.5 | 0.7 | 0.9 | 1.0 |
| 长音错误 | 0.3 | 0.5 | 0.7 | 1.0 | 1.0 |
| 清浊混淆 | 0.2 | 0.3 | 0.5 | 0.7 | 0.9 |
| 拨音错误 | 0.2 | 0.3 | 0.5 | 0.9 | 1.0 |
| 语调错误 | 0.1 | 0.2 | 0.3 | 0.5 | 0.7 |
| 声调错误 | 0.1 | 0.2 | 0.4 | 0.6 | 0.8 |

## 纠错数量限制（按水平分层）
初学者工作记忆容量有限，情感过滤较强，过多反馈会打击信心、阻碍吸收。

| 学生水平 | 最多纠错项数 | 说明 |
|----------|-------------|------|
| N5 | 2项 | 只纠正最严重的2个错误，保护学习信心 |
| N4 | 3项 | 纠正主要错误，兼顾鼓励与指导 |
| N3 | 4项 | 较全面纠错，学生已有一定吸收能力 |
| N2 | 5项 | 接近全面纠错，能接受较多反馈 |
| N1 | 不限 | 所有错误全部指出，追求精益求精 |

### 纠错优先级
当错误数量超过上限时，按以下优先级筛选：
1. 影响语义的错误（清浊、长音、促音）
2. 反复出现的高频错误
3. 语调错误

# 学生自适应学习系统

## 核心逻辑
系统根据学生发音错误、流畅度、成绩趋势，自动判断薄弱点、动态调整难度，并推荐最适合的练习内容。

## 自适应判断维度
1. 发音错误类型（促音/长音/清浊/拨音）
2. 错误频率与集中度
3. 流畅度稳定性
4. 水平进步速度

## 自适应练习推荐
| 主要错误类型 | 系统自动推荐练习 |
|--------------|------------------|
| 促音错误多 | 促音专项单词、句型朗读 |
| 长音错误多 | 长音专项单词、句子朗读 |
| 清浊混淆多 | 清浊对立词对比朗读 |
| 拨音错误多 | 拨音单词、句型练习 |
| 语调错误多 | 句型例句语调训练 |
| 流畅度偏低 | 慢速短句 → 逐步提升语速 |

## 自适应升降级规则
### 自动升档
- 最近3次成绩 ≥ 7.5
- 无严重发音错误
- 流畅度稳定提升
→ 自动建议升级

### 自动降档
- 最近2次成绩 < 6.0
- 基础性错误频繁
→ 自动建议降级巩固

### 维持当前等级
- 有进步但未达标
→ 继续当前水平，强化薄弱点

## 学生档案（可选功能）
适用于教师长期追踪学生进步的场景，非必选。默认关闭。

### 开启方式
```json
{
  "enable_student_profile": true
}
```

### 档案功能
- 自动创建/更新学生档案
- 记录历次成绩和错误类型
- 自适应判断升档/降档
- 生成个性化练习建议

### 升档判断逻辑
```python
def check_level_upgrade(profile):
    history = profile["submission_history"][-5:]
    if len(history) >= 3:
        recent_3 = history[-3:]
        all_high_score = all(s["score"] >= 7.5 for s in recent_3)
        no_major_error = all(len(s["weak_points"]) <= 2 for s in recent_3)
        
        if all_high_score and no_major_error:
            return {
                "should_upgrade": True,
                "from_level": profile["current_level"],
                "to_level": next_level(profile["current_level"]),
                "confidence": "高",
                "suggestion": f"建议{profile['student_name']}尝试{next_level(profile['current_level'])}水平练习"
            }
    return {"should_upgrade": False}
```

## 双模式使用规则

### 教师批改模式
上传：音频 + 课文/题目图片（或标准文本）+ 学生姓名 + 学生水平
输出：完整评分报告 + 发音纠错 + 班级汇总 + 自适应学习建议

### 学生自测模式
上传：仅音频 + 自选水平
输出：个人评分 + 发音纠错 + 练习建议 + 水平评估 + 自适应推荐
无需图片、无需姓名

## 触发条件
自动识别：
- 音频 + 图片（或文本）+ 姓名 → 教师批改模式
- 仅音频 → 学生自测模式

## 输入参数
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|----------|
| audio | 文件 | 是 | 单词/句子/课文/替换练习音频 |
| image | 文件 | 否 | 原文图片（教师模式推荐） |
| standard_text | 字符串 | 否 | 标准文本（优于图片OCR） |
| student_name | 字符串 | 否 | 学生姓名（教师模式必填） |
| student_level | 字符串 | 否 | N5/N4/N3/N2/N1（默认N5） |
| class_id | 字符串 | 否 | 班级编号 |

## 评分规则（统一10分制）

### 评分维度
| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 发音准确性 | 70% | 假名、清浊音、长音、促音、拨音 |
| 语调与节奏 | 10% | 句尾语调、升降调、停顿、节奏 |
| 流畅度 | 20% | 无卡顿、无重复、表达自然 |

### 最终总分档位
| 最终得分 | 评价 |
|----------|------|
| 8.0–10.0 | 优秀 |
| 7.0–7.9 | 良好 |
| 6.0–6.9 | 中等 |
| 6.0以下 | 待加强 |

## 替换练习特殊规则
- 教师需提供句型框架 + 允许替换词列表
- 系统只检查句型结构 + 替换词合法性 + 发音正确性
- 不将“替换词与原文不一致”视为错误

## 评测规范
评测原则:
- 鼓励为主，评分适度放宽
- 根据学生水平动态调整扣分力度
- 只纠正影响语义的明显发音错误
- 微小识别误差不予扣分
- Whisper转写结果仅供参考
- 纠错数量按水平分层限制

反馈要求:
- 语言：中文
- 纠错粒度：单词级或短语级
- 必须标注：原句、学生读法、标准读法、错误类型
- 文件编码：UTF-8

错误类型定义:
| 错误类型 | 示例 |
|----------|------|
| 促音错误 | がっこう → がこう |
| 长音错误 | おかあさん → おかさん |
| 清浊混淆 | びょういん → ひょういん |
| 拨音错误 | にほん → にほ |
| 语调错误 | 句尾升调读成降调 |
